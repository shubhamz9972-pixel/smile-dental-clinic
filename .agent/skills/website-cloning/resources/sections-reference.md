# Dental Website Sections Reference

Common sections found in dental clinic websites, ord by typical placement.

## 1. Navigation Bar
- Logo (left) + Menu items (center/right) + CTA button (far right)
- Sticky/fixed on scroll
- Hamburger menu on mobile
- Menu items: Home, Services, About, Doctors, Gallery, Contact
- CTA: "Book Appointment" or "Call Now"

## 2. Hero Section
- Full-width background image (happy patient or clinic interior)
- Large headline: clinic name or value proposition
- Subheadline: tagline or brief description
- Primary CTA button: "Book Appointment"
- Secondary CTA: "Call Now" or phone number
- Optional: Trust badges (years of experience, patient count)

## 3. Services Section
- Grid layout (2x3 or 3x4 cards)
- Each card: Icon + Service name + Short description
- Hover effect on cards
- Optional "Learn More" link per service
- Common services: Cleaning, Root Canal, Implants, Whitening, Braces, Crowns

## 4. About / Why Choose Us
- Doctor/clinic photo on one side
- Text content on the other side
- Highlights: Years of experience, modern equipment, gentle care
- Trust indicators: certifications, affiliations
- Optional: Statistics (patients served, procedures done)

## 5. Doctor Profiles
- Photo + Name + Qualifications
- Specialization and experience
- Short bio
- Can be a single profile or team grid

## 6. Testimonials
- Patient reviews in cards or carousel
- Star ratings
- Patient name (first name only for privacy)
- Quote text
- Optional: patient photo (with consent)

## 7. Gallery
- Grid of clinic photos
- Categories: Interior, Equipment, Team, Before/After
- Lightbox on click
- Responsive grid (2 cols mobile, 3-4 cols desktop)

## 8. Contact / Booking Section
- Contact form: Name, Phone, Email, Message, Preferred Date
- Clinic address with Google Maps embed
- Phone number (click-to-call)
- WhatsApp button
- Working hours table
- Email address

## 9. Footer
- Clinic name and brief description
- Quick links (Services, About, Contact)
- Contact info (phone, email, address)
- Social media icons (Facebook, Instagram, Google)
- Copyright notice
- Designed by credit (optional)
